A minimalist Daily & Weekly Tracker for gamers, optimized for mobile web. Keep track of your daily tasks, weekly bosses, and stamina at a glance.

## ✨ Features

- **Daily/Weekly Tracker:** Task lists with an automatic reset cycle (every morning at **05:00 AM** or every Monday).
- **Stamina Tracker:** Real-time energy regeneration calculation, including the estimated time for a full recharge.
- **Motivation Streak:** Visual reward (Flame icon) for completing all daily tasks without interruption. 🔥
- **Custom Task Sorting:** Individually sort your lists via Drag & Drop using an intuitive drag handle.
- **Native UX Feel:** Optimized app experience with text-selection protection, context-based cursors, and smooth tab switching.
- **Visual Feedback:** Gamified rewards featuring confetti effects upon reaching 100% progress. 🎉
- **Local Data Persistence:** All settings and progress are securely stored in the browser's LocalStorage.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Grid & Flexbox)
- **Logic:** Vanilla JavaScript (ES6+)
- **Libraries:** - [SortableJS](https://sortablejs.github.io/Sortable/) (Drag & Drop)
  - [Canvas Confetti](https://www.npmjs.com/package/canvas-confetti) (VFX)

## 🚀 Installation & Usage

1. Clone the repository or download the ZIP file.
2. Open `index.html` in your browser.
3. **Pro Tip:** For the best mobile experience, use "Add to Home Screen" in your mobile browser (Safari/Chrome) to use the tracker like a native app.
